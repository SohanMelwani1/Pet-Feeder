# Pet-Feeder
A pet feeder meant for small pets like hamsters
This uses a switch kind of mechanism to create an opening for food to fall into a bowl(directed) with manual controls. 
Which is convenient and easy for the owner to print out and stick it into place(adhesive maybe required if using more flexible material).

I have designed this product to be efficient as well as keeping it affordable to owners, while having a minimalist design.
